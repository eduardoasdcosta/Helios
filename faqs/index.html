Como funciona este sistema de voto?¶

O sistema de voto electrónico da Universidade de Lisboa é baseado num projeto open-source - Helios Voting System. Utiliza um sistema de encriptação homomórfica para assegurar a privacidade do voto. Esta adaptação requer que o eleitor esteja autenticado pelos sistemas da UL para poder votar. Após o eleitor rever o seu boletim de voto, é-lhe dado a escolha entre depositar o boletim na urna ou auditá-lo. Auditar o boletim de voto permite ver se as escolhas feitas pelo eleitor se encontram corretas. Após a auditoria do boletim, este é descartado e um novo boletim é gerado. Um eleitor pode auditar os boletins de voto as vezes que quiser até estar satisfeito para depositar o boletim na urna. Um eleitor pode também votar as vezes que quiser que apenas o último voto será contabilizado.

Não consigo votar numa eleicão¶

A eleição ainda não começou, a eleição já terminou ou não está autorizado a participar nesta. Consulte a lista de eleitores para saber quem está autorizado a votar. De seguida consulte os seus atributos para verificar que estes estão corretos. Caso estejam incorretos, deverá contactar o administrador da eleição para que este o possa adicionar manualmente.

O email presente nos meus atributos está errado¶

Poderá mudar o seu e-mail externo manualmente aqui para o email pretendido.

Como posso criar uma eleição?¶

Necessita de privilégios de administrador de eleição para isso. Contacte um administrador da sua instituição.

Não consigo mexer na eleição que criei¶

Provavelmente os seus privilégios de administrador de eleição expiraram. Contacte um administrador da sua instituição para lhe serem restituídos os privilégios.
